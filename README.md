# Voca Practice

Vocabulary performance practice app for the four announced task types.

## Contents

- Part 1 terms: 308
- Part 2 phrasal verb examples: 196
- Part 3 textbook examples: 76
- Part 4 quotes: 51

The app builds every practice question from `data/voca-data.js` and stores wrong answers with wrong counts in the browser's local storage.

## Run

Open `index.html` in a current Chrome, Edge, or other modern browser. No install step is required.
